ajaxUpload is a simple yet functional and highly customizable jQuery plugin that allows you to upload files asynchronously.
  • It's very simple to implement
  • It's highly customizable
  • It has a very flexible backend implementation

    You can store any file-related data coming from your backend script.

  • It uses CSS3 animations for a smoother experience.
  • It has a LESS markup that makes it much easier to stylize and modify

Try it

$(".ajaxupload").ajaxUpload(
{ fileTypes: ['jpg','jpeg','png','gif'], maxSize: 2048 }
);
